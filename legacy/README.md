# Legacy — RUT Verifier CLI

Esta carpeta contiene el código original del **script CLI monolítico** que precedió a la plataforma web full-stack (`backend/` + `frontend/`).

## Archivos

- `rut_verifier.py` — Script original de automatización con Playwright para verificación de RUT desde línea de comandos.
- `selectors.json` — Selectores CSS originales del portal MUISCA.

## Nota

Estos archivos se conservan únicamente como referencia histórica. La versión activa del proyecto se encuentra en los directorios `backend/` (API REST con FastAPI) y `frontend/` (interfaz web con Astro.js).
